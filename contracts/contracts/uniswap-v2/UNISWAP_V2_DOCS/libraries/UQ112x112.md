# libraries/UQ112x112.sol

## 文件职责

- 112.112 定点数库，用于 TWAP（时间加权平均价格）相关的价格累积计算。

## 核心概念

- `Q112 = 2**112`
- 通过乘以 `Q112` 把整数转换为定点数表示。

## 方法

- `encode(uint112 y) internal pure returns (uint224 z)`

  - `z = uint224(y) * Q112`。

- `uqdiv(uint224 x, uint112 y) internal pure returns (uint224 z)`
  - `z = x / y`。

## 被谁使用

- `UniswapV2Pair._update`：
  - 用 `UQ112x112.encode(reserve).uqdiv(otherReserve)` 计算价格并累加到 `priceXCumulativeLast`。
